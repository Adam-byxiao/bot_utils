## 界面概览

KWS GUI 界面已成功创建并可以正常启动。界面采用现代化的布局设计，分为以下几个主要区域：

## 使用流程演示

### 1. 启动应用
```bash
cd voicecommand_script
python start_gui.py
```

### 2. 配置连接
1. 在左上角输入设备信息：
   - IP地址: `192.168.1.100`
   - 用户名: `root`
   - 密码: `test0000`

### 3. 开始监控
1. 点击"开始监控"按钮
2. 观察右侧日志输出连接状态
3. 查看左下角统计数据实时更新

### 4. 模式切换
1. 在右上角选择输出模式：
   - **Standard**: 简洁模式，适合长时间监控
   - **Debug**: 详细模式，适合调试分析

### 5. 数据导出
1. 点击"导出数据"按钮
2. 选择保存位置
3. 生成包含以下字段的CSV文件：
   - 时间戳
   - 提示词
   - 识别分数
   - 是否触发
   - 开始时间
   - 结束时间
   - 原始日志

## 故障排除

### 常见问题
1. **GUI无法启动**
   - 检查wxPython是否正确安装
   - 确认Python版本兼容性

2. **连接失败**
   - 验证网络连接
   - 检查SSH服务状态
   - 确认认证信息正确

3. **数据显示异常**
   - 检查日志格式
   - 验证设备上的KWS服务
   - 查看Debug模式输出
